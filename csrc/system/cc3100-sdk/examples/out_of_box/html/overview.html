The SimpleLink CC3100 device is a network processor designed to interface with any microcontroller:

The CC3100 BoosterPack is a development platform to help you design your next Wi-Fi innovation with the CC3100 device.  It is a perfect companion to TI’s MSP430™ LaunchPad.
In addition to the LaunchPads, there is support available to mate the board with a FTDI Debug board to interface directly to a PC host using USB cable.
